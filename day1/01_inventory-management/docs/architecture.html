Factory Inventory Management

Full-stack demo application: a Vue 3 single-page app talking to a FastAPI backend that serves JSON-file-backed mock data. No database — everything is loaded into memory at startup and filtered in Python on each request.

Data Flow

Tracing a single user action: selecting a warehouse on the dashboard.

  1. User picks a warehouse in FilterBar; the shared filter ref (e.g. useFilters) updates to "San Francisco".
  2. Dashboard.vue reacts to the filter change and calls api.getDashboardSummary({ warehouse: "San Francisco" }).
  3. client/src/api.js issues an axios GET to /api/dashboard/summary?warehouse=San%20Francisco.
  4. FastAPI routes the request to the handler in server/main.py; query params are bound as typed arguments.
  5. apply_filters() filters the in-memory inventory_items and orders lists by warehouse.
  6. Handler computes KPIs: total inventory value, low-stock count, pending orders, backlog total, total order value.
  7. Pydantic validates the result against the response model and serializes it to JSON.
  8. Dashboard.vue stores the response in a ref; computed properties reformat numbers and the template re-renders the KPI cards.
